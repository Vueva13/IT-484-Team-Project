# Steps to build on Cloud 9

1. rvm install ruby-2.6.1

2. git clone https://github.com/Vueva13/IT-484-Team_Purple.git

3. cd it484_team_project

4. cd RideShare

5. bundle install

6. rake db:create

7. rake db:seed

8. rake db:migrate

9. rails server -p $PORT -b $IP
